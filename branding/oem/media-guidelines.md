# ArizenOS Media Guidelines

## Press and Media Usage

### Permitted
- Reproduce logos in articles, reviews, and tutorials about ArizenOS
- Use screenshots in editorial context
- Use the name "ArizenOS" to accurately describe the software

### Not Permitted
- Modify the logo in any way (color, shape, proportion)
- Imply endorsement or partnership without written approval
- Use ArizenOS branding for commercial products without a license
