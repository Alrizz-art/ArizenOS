# v0.1.0 — First Alpha

**Status:** In Development
**Target audience:** Contributors and invited testers
**Milestone:** [github.com/Alrizz-art/ArizenOS/milestone/1](https://github.com/Alrizz-art/ArizenOS/milestone/1)

---

## Scope

v0.1.0 is the minimum viable product of ArizenOS. The goal is not polish — it is to have every core component working end-to-end so contributors can build on a real foundation.

---

## Deliverables

### Core Packages (All required)

#### `@arizen/core` v0.1.0
- [ ] Logger with structured output, log rotation, and configurable log levels
- [ ] EventBus with typed publish/subscribe
- [ ] IPC bridge with typed channels and streaming support
- [ ] Config system with schema validation, merging, and file watching
- [ ] Base error types (`ArizenError` and core subclasses)
- [ ] Utility functions (`sleep`, `retry`, `debounce`, `throttle`, `generateId`)
- [ ] 100% unit test coverage (enforced by CI)
- [ ] Full JSDoc for all public APIs

#### `@arizen/shell` v0.1.0
- [ ] Win32 bindings: taskbar management, window enumeration, focus control
- [ ] DWM integration: thumbnail registration, composition state
- [ ] System tray: icon registration, context menu, click handling
- [ ] Windows startup: auto-start registration via registry
- [ ] DirectComposition: off-screen surface creation and management
- [ ] Shell mode: registration as default shell, restore-to-Explorer command
- [ ] Multi-monitor: monitor enumeration, per-monitor DPI

#### `@arizen/glass` v0.1.0
- [ ] DirectComposition + Direct2D blur rendering pipeline
- [ ] DWM thumbnail capture for real-time desktop content
- [ ] Depth layer rendering (Z0–Z4) with shadow compositing
- [ ] Hardware capability detection and fallback activation
- [ ] CSS fallback renderer for non-DirectComposition environments
- [ ] GlassEngine TypeScript API (mount, unmount, setBlurRadius, setDepth)
- [ ] Performance: < 5ms GPU frame time at High quality on RTX 3060

#### `@arizen/mind` v0.1.0
- [ ] llama.cpp N-API native binding compilation (CUDA, CPU-AVX2 targets)
- [ ] Model loading from `.gguf` files
- [ ] Streaming token generation (`generate()`, `chat()`)
- [ ] Token counting (`countTokens()`)
- [ ] Tokenization / detokenization
- [ ] GPU layer configuration and VRAM usage reporting
- [ ] ModelManager: load, unload, list, memory tracking
- [ ] AbortSignal support for cancellable inference

#### `@arizen/flow` v0.1.0
- [ ] MotionToken system tied to `@arizen/skin` duration/easing tokens
- [ ] Spring animation primitive (mass, stiffness, damping)
- [ ] Reduced motion detection and automatic fallback to instant transitions
- [ ] `animate()`, `spring()`, `sequence()` utilities
- [ ] React hooks: `useSpring`, `useAnimate`, `useReducedMotion`

#### `@arizen/skin` v0.1.0
- [ ] JSON theme file loading and schema validation
- [ ] Built-in themes: Void, Aurora, Ash, Prism
- [ ] Token compilation to CSS variables
- [ ] Theme application to Electron BrowserWindows
- [ ] `useTheme()` React hook
- [ ] Hot-reload in development mode
- [ ] WCAG 2.1 AA contrast audit utility

#### `@arizen/ui` v0.1.0
- [ ] Component set: Button, Input, Textarea, Select, Checkbox, Radio, Toggle
- [ ] Layout: Panel, GlassPanel, Divider, Stack, Grid
- [ ] Overlay: Modal, Drawer, Tooltip, Popover, ContextMenu, Toast
- [ ] Navigation: Tabs, Breadcrumb, Sidebar
- [ ] Display: Badge, Avatar, Tag, Spinner, ProgressBar
- [ ] All components: keyboard navigation, WCAG 2.1 AA, `useReducedMotion`-aware animations
- [ ] All components: dark + light mode via `@arizen/skin` tokens only (no hardcoded colors)
- [ ] Storybook catalog for all components

### Applications

#### Arizen Launcher v0.1.0-alpha
- [ ] Taskbar replacement: pinned apps, running app list, clock, system tray
- [ ] Spotlight launcher: app search, recent files, calculator, web search
- [ ] Virtual desktops: create, delete, switch, rename (enhanced Windows VD)
- [ ] Glass taskbar with real blur on all monitors
- [ ] Shell mode toggle in settings
- [ ] Multi-monitor: independent taskbar per monitor

#### Arizen Assistant v0.1.0-alpha
- [ ] Floating glass panel with conversation UI
- [ ] Streaming responses from local model via `@arizen/mind`
- [ ] Context sources: active window, clipboard, recent files
- [ ] Manual context: drag file, paste path, @clip, @screen
- [ ] Model selector (switch mid-conversation)
- [ ] Conversation history (local SQLite, AES-256 encrypted)
- [ ] Keyboard shortcut: Win + Space

#### Arizen Hub v0.1.0-alpha
- [ ] Theme browser and installer
- [ ] AI model browser and downloader (Hugging Face integration)
- [ ] Settings: glass quality, shell mode, keyboard shortcuts, startup
- [ ] System info: memory usage, loaded models, GPU status
- [ ] Keyboard shortcut: Win + Shift + H
