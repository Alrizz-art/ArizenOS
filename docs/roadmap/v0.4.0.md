# v0.4.0 — Public Beta

**Status:** Planned
**Target audience:** General public — first public release
**Prerequisite:** v0.1.0, v0.2.0, v0.3.0 all closed

---

## Scope

v0.4.0 is the first release intended for general public use. It is not v1.0.0 — it carries a beta label and some APIs are still evolving. But the core experience is polished enough that a non-technical user can install and use ArizenOS productively.

---

## Deliverables

### New Packages

#### `@arizen/sync` v0.4.0
- [ ] E2E encrypted sync of settings, themes, and conversation history
- [ ] Client-side AES-256-GCM encryption before any data leaves the device
- [ ] Server-side: encrypted blobs stored only — zero knowledge of content
- [ ] Conflict resolution: last-write-wins with 30-day conflict history
- [ ] Sync status UI in Hub
- [ ] Offline-first: all data available without network connection
- [ ] Manual sync trigger option (no forced background sync)

### Updated Packages

#### `@arizen/agent-sdk` — Public Preview Graduation
- [ ] API frozen for v0.4.0 (no breaking changes until v0.5.0 at earliest)
- [ ] Complete TypeDoc documentation published
- [ ] CLI tooling: `arizen create-extension`, `arizen pack`, `arizen publish`
- [ ] Extension sandbox security audit complete
- [ ] Extension signing with Arizen Developer certificates

#### `@arizen/mind` — Cloud Provider Support
- [ ] OpenAI API support (opt-in, API key required)
- [ ] Anthropic API support (opt-in, API key required)
- [ ] Provider priority order: local → cloud fallback (configurable)
- [ ] Clear UI indicator when cloud inference is active

### Updated Applications

#### Arizen Hub — Extension Marketplace
- [ ] Browse and install community extensions (Agent tools + widgets)
- [ ] Extension reviews, ratings, and download counts
- [ ] Extension signing verification (unsigned extensions show warning)
- [ ] Developer portal integration for extension publishing

#### Arizen Assistant — Enhancements
- [ ] Personas (named system prompts, switchable in session)
- [ ] Conversation search
- [ ] Response bookmarking (Hub → Saved Responses)
- [ ] Export conversation as Markdown
- [ ] Image input (multimodal models) where model supports it

#### Arizen Launcher — Polish
- [ ] Spotlight: file content search (`f:` prefix)
- [ ] Jump lists (right-click app for recent files)
- [ ] Taskbar grouping options (combine/separate/never)
- [ ] Context menus: run as admin, open in terminal, pin/unpin

## Quality Gates

| Gate | Requirement |
|---|---|
| User testing | Minimum 20 external beta testers complete structured UX evaluation |
| Crash rate | < 1 crash per 100 hours of usage across beta testers |
| Memory | ArizenOS shell idle RAM < 300 MB (without any model loaded) |
| Sync | Data encrypted before leaving device — verified by independent security review |
| Extension marketplace | All listed extensions pass automated security scan |
| Documentation | All user-facing features documented in docs/ |
| Installer | Uninstaller fully removes ArizenOS with no registry residue |
| Accessibility | All apps pass WCAG 2.1 AA audit |

---

## Breaking Changes from v0.3.x

- `@arizen/agent-sdk` tool permission API is finalised — any beta changes will have a migration guide
- Theme schema v1 is frozen — v2 (if needed) will have an automated migration tool
- Config file format locked — TOML keys will not change without a major version

## Migration Guide

A detailed migration guide from v0.3.x to v0.4.0 will be published at [docs.arizenos.dev/migration/v0.4](https://docs.arizenos.dev/migration/v0.4) when the release is finalised.
